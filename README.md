# ✈️ Flight Delays Analysis – Tableau Dashboard

## 📌 Project Overview

A complete **flight delays analysis** project using Tableau Public. The goal was to identify **what causes delays**, **which airlines/airports are worst affected**, and **compare arrival vs departure delays**.

---

## 🎯 What I Built

### 1️⃣ What Causes Delays?
- Analyzed **14 airlines** and focused on **top 3 airlines with biggest delays**
- Discovered: **Late aircraft delay** causes the most delays
- **Security delays** cause the least

### 2️⃣ Arrival vs Departure Delays
- Compared delays at **IAH** and **JFK** airports
- **JFK** has significantly higher average departure delay (31.02) vs IAH (2.99)

### 3️⃣ Maximum & Minimum Delays
- **Late aircraft delays** → highest delay counts (15K+ flights)
- **Security delays** → lowest delay counts

### 4️⃣ Worst Airlines & Airports
- Identified destinations and arrival airports with most delays
